Green Notez

I created Green Notez to help keep track of water changes, daily checks and problems for hydroponic gardening. I felt most other grow tracking apps were too complicated and gave the user too much to work with. I created this app for a more user friendly experience that keeps record of the most important data and allows the user to share this information. This project was created using React and Supabase for the database storage and user authentication. Users can create .jpg note cards that display their data to share online to ask for help, show off or to save for personal use.

Green Capz

I created Green Capz to help monitor the reservoir data in a hydroponic garden. This app reads data from a MongoDB database and displays a users current temperature, humidity and water PPM. User's data from the past week is also displayed on line charts. I created this project using a Node/Express/MongoDB/Pug stack. I used express-sessions for user sessions and bcrypt to store user passwords and authenticate. Creating the authentication and session data was my most difficult task and took many Google searches to set it up.

Green Capz Sensor Controller

I created the sensor controller for the Green Capz app using a Raspberry Pi connected to an Arduino board. The Arduino board reads analog sensors, converts to digital signal and is sent to the Raspberry Pi. The Raspberry Pi is running an Express/Node/MongoDB/Pug stack which saves user data to be accessed from Green Capz. The box containing the Raspberry Pi and Arduino boards has a mounted display to show current readings. The Arduino board needed minimal C code written but was an enjoyable learning experience to dabble in another programming language.
