Green Notez

Green Notez is a user-friendly app I created to track hydroponic gardening data, including water changes, daily checks, and problems. Unlike other complicated grow tracking apps, Green Notez focuses on recording the most important data and enables users to share this information via visually-appealing .jpg note cards. Built using React and Supabase for database storage and user authentication, Green Notez provides a streamlined experience for users to record and monitor their hydroponic garden.

Green Capz

Green Capz is an app I developed to monitor hydroponic garden reservoir data, including temperature, humidity, and water PPM. Using a Node/Express/MongoDB/Pug stack, Green Capz reads data from a MongoDB database and presents users with their current and past week's data on line charts. To enable user authentication and session data, I utilized express-sessions and bcrypt for secure password storage. Implementing authentication and session management proved challenging, requiring extensive online research to ensure optimal functionality.

Green Capz Sensor Controller

For the Green Capz app, I developed a sensor controller using a Raspberry Pi and Arduino board. The Arduino board reads analog sensors, converts the data into a digital signal, and transmits it to the Raspberry Pi. The Raspberry Pi, running on an Express/Node/MongoDB/Pug stack, stores user data for access from Green Capz. To provide users with real-time readings, I mounted a display on the box housing the Raspberry Pi and Arduino boards. Although minimal C code was needed for the Arduino board, this project offered an opportunity to learn a new programming language and further expand my skillset.
